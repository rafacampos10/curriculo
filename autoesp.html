El vehículo autónomo se mueve en un entorno no estructurado, realizando movimientos autónomos, conduciendo de un punto a otro en el entorno automáticamente. El vehículo utiliza sensores ultrasónicos para medir la distancia de posibles obstáculos, pudiendo desviarse de ellos de forma autónoma, evitando así colisiones. Para la implementación del software, se utilizó la plataforma informática integrada del tipo Arduino.
Se probaron diferentes algoritmos para identificar y evitar obstáculos. Cada uno de los procedimientos de aprendizaje muestra un tipo diferente de mecanismo evolutivo. Los experimentos de simulación mostraron que el sistema para evitar obstáculos es capaz de aprender estrategias de navegación en entornos desconocidos, con un costo y tiempo de cálculo reducidos.

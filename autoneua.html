The autonomous vehicle moves in an unstructured environment, making autonomous movements, driving from one point to another in the environment automatically. The vehicle uses ultrasonic sensors to measure the distance from possible obstacles, being able to deviate from them autonomously, thus avoiding collisions. For the software implementation, the embedded computer platform of the Arduino type was used.
Different algorithms were tested to identify and avoid obstacles. Each of the learning procedures shows a different type of evolutionary mechanism. Simulation experiments showed that the obstacle avoidance system is able to learn navigation strategies in unknown environments, with reduced cost and computational time.
